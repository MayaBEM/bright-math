Key terms — Chapter 2

ExpressionA mathematical phrase with numbers, letters and operations, but no equals sign.
FormulaA rule with an equals sign that shows how one quantity is calculated from others.
EquationA statement with an equals sign that can be solved to find one unknown value.
VariableA letter that represents a number which can change.
CoefficientThe number multiplying a variable, e.g. the 4 in 4x.
ConstantA fixed number in an expression with no letter attached.
TermOne part of an expression, separated by + or − signs.
SubstituteTo replace a variable with a given number.
EvaluateTo calculate the numerical value of an expression.
Like termsTerms that share exactly the same letter part, e.g. 3x and 7x.
